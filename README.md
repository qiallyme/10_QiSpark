# 10_QiSpark

QiSpark serves as the static documentation engine and cockpit launcher for the QiLabs ecosystem.

## Directory Structure

- **`00_config/`**: Configuration files and templates for document compilation.
- **`10_site/`**: Generated static website outputs and web assets.
- **`20_snapshots/`**: Captured system snapshots and state exports.
- **`30_ops/`**: Operational runbooks and execution scripts.
- **`90_archive/`**: Archived history and legacy documents.
