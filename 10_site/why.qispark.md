# Why QiSpark?

QiSpark serves as the static rendering engine and user interface portal (cockpit launcher) for the QiLabs ecosystem. It isolates public/publishable documentation from the secure raw workspace data, providing a performant, modern interface while maintaining a secure boundaries posture.
