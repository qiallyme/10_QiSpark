    Main System Flow

    REAL LIFE
  ↓
What Happened?
  ↓
QiBit Capture
  ↓
Triage / Interpret
  ↓
Bucket + Thread + People + Meaning
  ↓
Action Required?
  ├── No → Save as Knowledge / Log / Reference
  └── Yes → Create Action → Add Steps → Slot in Time
  ↓
Work / Waiting / Done
  ↓
Resolution
  ↓
Reflection
  ↓
Retrieval through Timeline / Search / AI
